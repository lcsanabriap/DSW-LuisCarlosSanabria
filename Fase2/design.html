Dar a conocer desde la mano de la UNAD el aprendizaje y usos de HTML en Etiquetas semánticas Y CSS Grid

La finalidad de este sitio es dar a entender la funcionalidad de las etiquetas semanticas y CSS Grib dentro de la estructura de un documento HTML y su importancia dentro de los motores de busqueda para que con exactitud nos indiquen qué partes de su contenido corresponden a cada una de las partes típicas de un sitio web. CSS Grib es un sistema de posicionamiento para elementos de una pagina web cuando trabajamos en una estructura de codigo como HTML definimos la grilla especificando cuantas columnas y cuantas filas tendra la grilla (espacio Grafico) para llenarlo de contenido.
