Dar a conocer desde la mano de la UNAD el aprendizaje y usos de HTML en Etiquetas Semánticas Y CSS Grid.

La finalidad de este sitio es dar a entender la funcionalidad de las etiquetas semanticas y CSS Grib dentro de la estructura de un documento HTML y su importancia dentro de los motores de busqueda para que con exactitud nos indiquen qué partes de su contenido corresponden a cada una de las partes típicas de un sitio web.

Etiquetas Semanticas

las etiquetas semanticas nos sirven como un mecanismos automatico como lo es un navegador web nos indiquen con exactitud a que parte de contenido corresponden a ese sitio web, las etiquetas semanticas estructurales como "section" utilizada para representar secciones o subsecciones acompañadas de uni h1 -h6 que le dan dimencion a las letras otras etiquetas como article, aside, footer, hgroup, entre otras mas.

CSS Grib

CSS Grib es un sistema de posicionamiento para elementos de una pagina web cuando trabajamos en una estructura de codigo como HTML definimos la grilla especificando cuantas columnas y cuantas filas tendra la grilla o (espacio Grafico) para llenarlo de contenido.
